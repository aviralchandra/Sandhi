00078       /*!
00079        * Set the channel map. Channels are numbers as:
00080        *     N/2+1 | ... | N-1 | 0 | 1 |  2 | ... | N/2
00081        *    <------------------- 0 -------------------->
00082        *                        freq
00083        *
00084        * So input stream 0 goes to channel 0, etc. Setting a new channel
00085        * map allows the user to specify where in frequency he/she wants

00087        * putting signals into the channels on the edge of the spectrum
00088        * which can either wrap around (in the case of odd number of
00089        * channels) and be affected by filter rolloff in the transmitter.
00090        *
00091        * The map must be at least the number of streams being sent to the
00092        * block. Less and the algorithm will not have enough data to
00093        * properly setup the buffers. Any more channels specified will be
00094        * ignored.
00095        */
